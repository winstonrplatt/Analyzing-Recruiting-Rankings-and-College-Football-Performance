Overview

This simplified showcase demonstrates an analysis of how college football recruiting rankings relate to on-field performance. The original project was done in partnership with Nate Gordon in Fall 2024, but has since been updated.

df <- df %>%
  mutate(
    recruiting_rank = as.numeric(recruiting_rank),
    total_wins = as.numeric(total_wins),
    talent_rating = as.numeric(talent_rating),
    win_pct = total_wins / total_games
  ) %>%
  filter(!is.na(recruiting_rank) & !is.na(win_pct), recruiting_rank > 0)

The plots below highlight the relationship between recruiting rank and win percentage.

# Scatterplot with smoothing line (lower rank is better)
ggplot(df, aes(x = recruiting_rank, y = win_pct, label = team)) +
  geom_point(alpha = 0.6, size = 2) +
  geom_smooth(method = "lm", se = TRUE, color = "black", linetype = "dashed") +
  scale_x_reverse() + # if lower recruiting rank = better, reverse axis for readability
  labs(
    title = "Recruiting Rank vs Win Percentage",
    subtitle = "Lower recruiting rank indicates higher-rated recruiting class",
    x = "Recruiting Rank (lower is better)",
    y = "Win Percentage"
  ) +
  theme_minimal(base_size = 12)

Regression & Key Findings

Below is a brief linear model quantifies the predictive relationship between recruiting rank and win percentage.

# Simple linear model (win_pct ~ recruiting_rank)
model <- lm(win_pct ~ recruiting_rank, data = df)
